## <div id="1">1. Типы данных</div>
Числовые типы: `int`, `float`, `complex`.
Логические типы: `bool`.
Последовательности: `list`, `tuple`, `range`.

#### Общие операции с последовательностями
Операция | Результат
`x in s` | True, если элемент s равен x, иначе False
`x not in s` | False если элемент s равен x, иначе True
`s + t` | объединение s и t
`s * n` | эквивалентно добавлению s к самому себе n раз
`s[i]` | i- й элемент s , происхождение 0
`s[i:j]` | кусочек s от i до j
`s[i:j:k]` | кусочек s от i до j с шагом k
`len(s)` | длина с
`min(s)` | наименьший предмет из s
`max(s)` | самый большой предмет из s
`s.index(x[, i[, j]])` | индекс первого вхождения x в s (в или после индекса i и перед индексом j)
`s.count(x)` | общее количество вхождений x в с

#### Операции для изменяемых типов последовательностей
Операция | Результат
`s[i] = x` | элемент i из s заменяется на x
`s[i:j] = t` | фрагмент s от i до j заменяется содержимым итерируемого t
`del s[i:j]` | такой же как s[i:j] = []
`s[i:j:k] = t` | элементы s[i:j:k] заменяются элементами t
`del s[i:j:k]` | удаляет элементы s[i:j:k] из списка
`s.append(x)` | добавляет x в конец последовательности (так же, как `s[len(s):len(s)] = [x]`)
`s.clear()` | удаляет все элементы из s (так же, как `del s[:]`)
`s.copy()` | создает неглубокую копию s (так же, как `s[:]`)
`s.extend(t)` или `s += t` | расширяет s содержимым t (по большей части то же самое, что и s[len(s):len(s)] = t)
`s *= n` | обновляет s, его содержимое повторяется n раз
`s.insert(i, x)` | вставляет x в s перед индексом i (так же, как s[i:i] = [x])
`s.pop()` или `s.pop(i)` | извлекает элемент в i, а также удаляет его из s
`s.remove(x)` | удалить первый элемент из s, где s[i] = x
`s.reverse()` | меняет местами элементы s на месте

### <div id="1_1">1.1. Целое число (int)</div>
Тип int реализует абстрактный базовый класс numbers.Integral.
#### Побитовые операции с целочисленными типами по возрастанию приоритета:
Операция | Результат
`x | y` | поразрядно или по x и y
`x ^ y` | побитовое исключающее или из x и y
`x & y` | поразрядно и по x и y
`x << y` | x сдвинут влево на n бит
`x >> y` | x сдвинут вправо на n бит
`~x` | биты x перевернуты


#### Дополнительные методы для целочисленных типов:
##### int.bit_length()
Возвращает количество битов целого числа в двоичном формате, исключая знак и ведущие нули.
##### int.bit_count()
Возвращает количество единиц в двоичном представлении абсолютного значения целого числа.
##### int.to_bytes(length=1, byteorder='big', *, signed=False)
Возвращает массив байтов, представляющий целое число. length - длина в байтах, byteorder - порядок байтов ('big' - наиболее значимый байт в начале массива байтов, 'little' - наоборот), signed - используется ли дополнение до двух для представления целого числа.
##### int.from_bytes(bytes, byteorder='big', *, signed=False)
Возвращает целое число, представленное заданным массивом байтов.
##### int.as_integer_ratio()
Возвращает пару целых чисел, отношение которых равно исходному целому числу и имеет положительный знаменатель.
##### int.is_integer()
Возвращает True.


### <div id="1_2">1.2. Число с плавающей точкой (float)</div>
Тип float реализует абстрактный базовый класс numbers.Real.

#### Дополнительные методы для целочисленных типов:
##### float.as_integer_ratio()
Возвращает пару целых чисел, отношение которых точно равно исходному числу с плавающей запятой.
##### float.is_integer()
Является ли целым числом.
##### float.hex()
Возвращает представление числа в виде шестнадцатеричной строки.
##### float.fromhex(s)
Метод класса для возврата числа с плавающей запятой, представленного шестнадцатеричной строкой s.


### <div id="1_3">1.3. Комплексное число (complex)</div>
Комплексные числа имеют действительную и мнимую часть, каждая из которых представляет собой число с плавающей запятой. Чтобы извлечь эти части из комплексного числа z, используйте z.real и z.imag.


### <div id="1_4">1.4. Логический тип (bool)</div>
Логические значения представляют значения истинности True или False. Является подклассом int.


### <div id="1_5">1.5. Список (list)</div>
__Список__ - изменяемая последовательность для хранения коллекций однородных элементов. 
##### Способы создания списка
`[]`, `[a, b, c]`, `[x for x in iterable]`, `list()`.
#### Дополнительные методы списков
##### sort(*, key=None, reverse=False)
Сортирует список. `key` - функция одного аргумента для извлечения ключа сравнения из каждого элемента списка, `reverse` - производить ли обратную сортировку.


### <div id="1_6">1.6. Кортеж (tuple)</div>
__Кортеж__ - неизменяемая последовательность для хранения коллекций разнородных данных.
#### Способы создания кортежа
`()`, `a,`, `(a,)`, `a, b, c,`, `(a, b, c)`, `tuple()`.


### <div id="1_7">1.7. Диапазон (range)</div>
__Диапазон__ - неизменяемая последовательность чисел для повторения определенного количества циклов for. Преимущество перед `list` и `tuple` в том, что в памяти хранятся только значения `start`, `stop`, `step`.
#### Способы создания диапазона
`range(start=0, stop, step=1)`. 


### <div id="1_8">1.8. Строка (str)</div>
__Строка__ - неизменяемая последовательность кодовых символов Юникода.
#### Способы создания строки
`'string'`, `"string"`, `"""string"""`.
#### Дополнительные методы строк
##### str.capitalize()
Возвращает копию строки с первым символом в верхнем регистре и остальными в нижнем регистре.
##### str.casefold()
Возвращает копию строки в нижнем регистре.
##### str.center(width, fillchar)
Возвращает отцентрированную строку длиной width с заполненым пространством символами fillchar.
##### str.count(sub, start, end)
Возвращает количество непересекающихся вхождений подстроки sub в диапазоне start - end.
##### str.encode(encoding='utf-8', errors='strict')
Возвращает строку закодированную в bytes. `errors` - способы обработки ошибок (`strict` - возвращает исключение UnicodeError; `ignore`, `replace`, `xmlcharrefreplace`, `backslashreplace` - подробнее `codecs.register_error()`).
##### str.endswith(suffix, start, end)
Возвращает `True`, если строка заканчивается `suffix` строкой или кортежем, иначе `False`. `start`, `end` определяют диапазон.
##### str.expandtabs(tabsize=8)
Возвращает копию строки, в которой все символы табуляции заменены одним или несколькими пробелами в зависимости то текущего столбца и tabsize.
##### str.find(sub, start, end)
Возвращает индекс первого совпадения `sub` в `str[start:end]`, иначе `-1`.
##### str.format(*args, **kwargs)
Выполняет форматирование строки по фигурным скобкам. Каждое поле замены содержит либо индекс аргумента, либо имя аргумента. Возвращает копию строки.
##### str.format_map(mapping)
Аналогично `str.format()`, но используется `mapping`, а не копируется в `dict`. Полезно, если `mapping` это подкласс `dict`.
##### str.index(sub, start, end)
Аналогично `find()`, но вызывает `ValueError`, когда sub не найдено.
##### str.isalnum()
Возвращает `True`, если символы строки являются буквами или цифрами и есть хотя бы один символ, иначе False. Символ является буквенно-цифровым, если верно одно из: `c.isalpha()`, `c.isdecimal()`, `c.isdigit`, `c.isnumeric()`.
##### str.isalpha()
Возвращает `True`, если все символы строки являются буквами и есть хотя бы один символ, иначе `False`. Буквами являются символы, которые имеют общее свойство Юникода одно из: `Lm`, `Lt`, `Lu`, `Ll`, `Lo`.
##### str.isascii()
Возвращает `True`, если строка пустая или все символы являются ASCII, иначе `False`.
##### str.isdecimal()
Возвращает `True`, если все символы в строке являются десятичными символами и есть хотя бы один символ, иначе `False`. Десятичный символ - символ общей категории Юникода `Nd`.
##### str.isdigit()
Возвращает `True`, если все символы в строке являются цифрами и есть хотя бы один символ, иначе `False`.
##### str.isidentifier()
Возвращает `True`, если строка является допустимым индентификатором, иначе `False`.
##### str.islower()
Возвращает `True`, если все символы строки нижнего регистра и есть хотя бы один символ, иначе `False`.
##### str.isnumeric()
Возвращает `True`, если все символы строки являются числами и есть хотя бы один символ, иначе `False`.
##### str.isprintable()
Возвращает `True`, если все символы строки печатыемые или строка пустая, иначе `False`. Непечатаемые символы - это символы Юникода «Другие» или «Разделители», за исключением пробела ASCII (0x20), который считается печатаемым.
##### str.isspace()
Возвращает `True`, если в строке только пробельные символы и есть хотя бы один символ, иначе `False`.
##### str.istitle()
Возвращает `True`, если символы верхнего регистра следуют за символами без регистра, а символы нижнего регистра следуют только за символами с регистром и содержит хотя бы один символ, иначе `False`.
##### str.isupper()
Возвращает `True`, все символы в строке имеют верхний регистр и есть хотя бы один символ, иначе `False`. 
##### str.join(iterable)
Возвращает строку, которая добавляет в строку `iterable`, иначе `TypeError`.
##### str.ljust(width, fillchar)
Возвращает строку длиной `width` с выравниванием по левому краю, заполняя пространство символами `fillchar`.
##### str.lower()
Возвращает копию строки нижнего регистра.
##### str.lstrip(chars)
Возвращает копию строки с удаленными начальными символами. `chars` - набор удаляемых символов, по умолчанию пробел.
##### static str.maketrans(x, y, z)
Возвращает таблицу перевода для `str.translate()`. Если передан один аргумент, то это должен быть словарь, сопоставляющий порядковые номера Юникода (целые числа) или символы (строки длины 1) с порядковыми номерами Юникода, строками (произвольной длины) или `None`. Если передано два аргумента, то они должны быть строками одинаковой длины, и в результирующем словаре каждый символ x будет сопоставлен с символом в той же позиции в y. Если есть третий аргумент, это должна быть строка, символы которой будут сопоставлены Noneв результате.
##### str.partition(sep)
Разделяет строку при первом появлении `sep` и возвращает тройной кортеж, содержащий часть перед разделителем, сам разделитель и часть после разделителя. Если разделитель не найден, верните тройной кортеж, содержащий саму строку, за которой следуют две пустые строки.
##### str.removeprefix(prefix, /)
Возвращает `string[len(prefix):]`, если строка начинается с `prefix`, иначе копию исходной строки.
##### str.removesuffix(suffix, /)
Возвращает `string[:-len(prefix)]`, если строка заканчивается с `prefix`, иначе копию исходной строки.
##### str.replace(old, new, count)
Возвращает копию строки, заменяя `old` на `new` `count` количества раз.
##### str.rfind(sub, start, end)
Возвращает наибольший индекс в строке, где найден `sub` в диапазоне `[start:end]`, иначе `-1`.
##### str.rindex(sub, start, end)
Аналогично `rfind()`, но вызывает `ValueError`, если `sub` не найдено.
##### str.rjust(width, fillchar)
Возвращает строку, длиной `width`, выровненную по правому краю, заполняя простанство `fillchar`.
##### str.rpartition(sep)
Разделяет строку по последнему вхождению `sep` и возвращает тройной кортеж. Схоже с `str.partition(sep)`.
##### str.rsplit(sep=None, maxsplit=-1)
Возвращает список слов в строке, используя `sep` в качестве строки-разделителя. Если задан `maxsplit`, выполняется не более `maxsplit` разбиений, крайних правых. Если `sep` не указан, любая строка пробелов является разделителем. Схоже с `str.split()`.
##### str.rstrip(chars)
Возвращает копию строки с удаленными конечными символами. Схоже с `str.strip(chars)`.
##### str.split(sep=None, maxsplit=-1)
Возвращает список слов в строке, используя `sep` в качестве разделителя. Если задан `maxsplit`, будет выполнено не более `maxsplit` разбиений (таким образом, список будет содержать не более `maxsplit+1 `элементов). Если `maxsplit` не указан, то ограничений на количество разделений нет (выполняются все возможные разделения).
##### str.splitlines(keepends=False)
Возвращает список строк в строке, разрывающихся на границах строк. Разрывы строк включаются в список, если `keepends=True`. В отличие от `split()` возвращает пустой список для пустой строки, а разрыв конечной строки не приводит к появлению дополнительной строки.
##### str.startswith(prefix, start, end)
Возвращает `True`, если строка начинается с `prefix` строки или кортежа, иначе `False`. `start`, `end` определяют диапазон.
##### str.strip(chars)
Возвращает копию строки с удаленными начальными и конечными символами. `chars` - набор удаляемых символов, по умолчанию пробел.
##### str.swapcase()
Возвращает копию строки с символами верхнего регистра в нижнем регистре и наоборот.
##### str.title()
Возвращает копию строки, где слова начинаются с верхнего регистра, а последующие символы нижнего регистра.
##### str.translate(table)
Возвращает копию строки, где каждый символ сопоставлен с помощью таблицы перевода `table`.
##### str.upper()
Возвращает копию строки в верхнем регистре.
##### str.zfill(width)
Возвращает копию строки, заполненную '0', чтобы создать строку длины width, иначе исходную строку.


### <div id="1_9">1.9. Байты (bytes, bytearray, memoryview)</div>
`bytes` и `bytearray` поддерживают строчные методы.
#### bytes
`bytes` - неизменяемые последовательности отдельных байтов. Байтовые литералы поддерживают только символы ASCII. Две шестандцатеричные цифры соответствуют одному байту.
#### Способы создания bytes
`b'string'`, `b"string"`, `b'''string'''`, `bytes(10)`, `bytes(range(20))`.
#### Дополнительные методы bytes
##### classmethod fromhex(string)
Возвращает обьект `bytes`, декодируя `string`. `string` должна содержать две шестандцатеричные цифры на байт, пробелы игнорируются.
##### hex(sep, bytes_per_sep)
Возвращает строковый обьект, содержащий две шестандцатеричные цифры для каждого байта. `sep` - разделитель, `bytes_per_sep` - управляет интервалом (положительные значения вычисляют положение справа, отрицательные - слева).
#### bytearray
`bytearray` - изменяемый аналог `bytes`, поддерживает изменяемые операции с последовательностями.
#### Способы создания bytearray
`bytearray()`, `bytearray(10)`, `bytearray(b'Hi')`.
#### Дополнительные методы bytearray
##### classmethod fromhex(string)
Возвращает обьект `bytearray`, декодируя `string`. `string` должна содержать две шестандцатеричные цифры на байт, пробелы игнорируются.
##### hex(sep, bytes_per_sep)
Возвращает строковый обьект, содержащий две шестандцатеричные цифры для каждого байта. `sep` - разделитель, `bytes_per_sep` - управляет интервалом (положительные значения вычисляют положение справа, отрицательные - слева).
#### memoryview
`memoryview` - обьект, который обеспечивает доступ к буферу памяти с данными. Позволяет работать с данными в памяти без копирования. Эффективно при обработке больших данных (изображения, аудио и т.д.).
#### Способы создания memoryview
`memoryview(b'abc')`
#### Дополнительные методы memoryview
##### __eq__(exporter)
Эквивалентно, если все соответствующие значения равны.
##### tobytes(order='C')
Возвращает данные в буфер в виде байтовой строки. Эквивалентно `bytes()`.
##### hex([sep[, bytes_per_sep]])
Возвращает строковый обьект, содержащий две шестандцатеричные цифры для каждого байта в буфере.
##### tolist()
Возвращает данные в буфер в виде списка элементов.
##### toreadonly()
Возвращает версию обьекта, доступную только для чтения.
##### release()
Освобождает базовый буфер. Помогает снять запрет на изменение размера и т.д.
##### cast(format[, shape])
Приводит представление в памяти к новому формату или форме.
##### obj
Базовый обьект.
##### nbytes
Обьем пространства в байтах для использования в непрерывном представлении.
##### readonly
Доступна ли память только для чтения.
##### format
Формат для каждого элемента представления.
##### itemsize
Размер в байтах каждого элемента.
##### ndim
Количество представлений памяти измерениями многомерного массива.
##### shape
Кортеж целых чисел, длина которого определяет форму памяти в виде массива.
##### strides
Кортеж целых чисел, длина которого определяет размер в байтах для доступа к каждому элементу для каждого измерения массива.
##### suboffsets
Используется внутри массивов в стиле PIL.
##### c_contiguous
Является ли память C непрерывной.

## <div id="2">2. Генератор</div>
__Генератор__ - это функция или выражение, которое позволяет создавать итерируемые последовательности значений по мере их необходимости, вместо того, чтобы заранее создавать их все и хранить в памяти. Существует два основных типа генераторов: функциональные генераторы (с использованием ключевого слова yield) и генераторные выражения.

Пример:
```python
def my_generator():
    yield 1
    yield 2

gen = my_generator()
print(next(gen)) # Вывод: 1


gen_expr = (x ** 2 for x in range(5))
print(list(gen_expr)) # Вывод: [0, 1, 4, 9, 16]
```


## <div id="3">3. Итератор</div>
__Итератор__ - обьект, представляющий поток данных. Итератор должен реализовать два метода: `__iter__()` - возвращает обьект итератора; `__next__()` - возвращает следующий элемент из итератора или `StopIteration`, если элементов нет.

Пример:
```python
class SimpleIterator:
    def __init__(self, start, end):
        self.current = start
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        if self.current < self.end:
            result = self.current
            self.current += 1
            return result
        else:
            raise StopIteration

my_iterator = SimpleIterator(1, 5)

for value in my_iterator:
    print(value)
```

### decimal
Модуль с Decimal типом данных для десятичной арифметики с плавающей запятой. Полезно для финансовых вычислений, точности десятичного представления, контроля округления, отслеживания значимых десятичных знаков:
```python
>>> from decimal import *
>>> round(Decimal('0.70') * Decimal('1.05'), 2)
Decimal('0.74')
>>> round(.70 * 1.05, 2)
0.73
```
